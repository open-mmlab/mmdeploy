# mmction2 模型支持列表

[MMAction2](https://github.com/open-mmlab/mmaction2)是一款基于 PyTorch 的视频理解开源工具箱，是[OpenMMLab](https://openmmlab.com)项目的成员之一。

## 安装 mmaction2

请参考[官网安装指南](https://github.com/open-mmlab/mmaction2#installation).

## 模型支持列表
